Detailed Description

Driver public API.

All API calls require a handle (cs40l25_t *) to the driver state. All API calls return a status

See also
CS40L25_STATUS_
Attention
The process() API must be called in either (1) a baremetal superloop or (2) an RTOS task loop in order to execute the following APIs:
  • control()
  • reset()
  • boot()
  • power()
  • calibrate()
