Detailed Description

Status of HALO FW.

List of registers can be accessed via status values, or indexed via words (when reading via Control Port). These fields are read multiple times to determine statuses such as is_hb_inc and is_temp_changed.

Warning
The list of registers MUST correspond to the addresses in cs40l25_dsp_status_addresses.
