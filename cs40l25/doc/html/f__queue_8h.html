typedef bool(* f_queue_copy) (void *from, void *to)

Abstract method for copying from one queue element type to another.

Since the queue copies contents into/out of the queue when inserting/removing, the initializer of the queue needs to provide the implementation for copying contents.

Parameters
[in]frompointer to element to copy contents from
[in]topointer of empty element to which the contents should be copied
Returns
success of copy (true = successful, false = failed)
